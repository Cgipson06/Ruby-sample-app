== Ruby on Rails Sample App 

The Rails sample app from Michael Hartl's Railstutorial.org
The application includes examples of the following major features of Rails:



* Models, Views and Controllers

* Templates and partials

* Filters, Validations and Callbacks

* has_many/belongs_to and has_many :through associations

* Security and testing
